# Leaderboard App

This project marks my first dive into **JavaScript**. It is a dynamic web application designed to manage and display scores, focusing on interactivity and DOM manipulation.

## 🚀 Project Overview
The Leaderboard App allows users to see a list of participants and their scores. This project helped me transition from static layouts to functional, logic-driven web development.

## 🛠️ Built With
- **HTML5 & CSS3:** For the application structure and styling.
- **JavaScript (Vanilla JS):** Used for dynamic content rendering and score management.
- **Modular Logic:** Structured with `ScoreCard.js` to keep the code organized.

## 📂 Features
- **Dynamic Updates:** The UI updates based on the underlying data.
- **Score Display:** Clean and organized visualization of leader data.
- **Modular Code:** Separated logic into different files to practice better code structure.

## 🧠 Key Learnings
While building my first JS app, I focused on:
1. **Selecting Elements:** Using `querySelector` and `getElementById`.
2. **Event Handling:** Making the page responsive to user actions.
3. **DOM Manipulation:** Creating and injecting HTML elements dynamically via JavaScript.
4. **Data Handling:** Managing simple arrays or objects to represent player statistics.
